# BrightDeveloper

**The Complete Guide to BrightSign Development**

Welcome to BrightDeveloper, your comprehensive resource for developing applications on BrightSign digital signage players. Whether you're writing BrightScript, JavaScript, building native extensions, or integrating with BSN.cloud, this guide has you covered.

---

## Quick Start

New to BrightSign development? Start here:

1. [Introduction to BrightSign Players](part-1-getting-started/01-introduction-to-brightsign-players.md) - Understand the hardware platform
2. [BrightScript Language Reference](part-2-brightscript-development/01-brightscript-language-reference.md) - Learn the native scripting language
3. [Hello World Example](part-2-brightscript-development/examples/foundations/01-hello-world.brs) - Run your first program
